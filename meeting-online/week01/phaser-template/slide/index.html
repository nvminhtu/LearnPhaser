1- Normal Package

Module Pattern

Xem thêm tại Mastering Module Pattern

						
								var Module = (function () {

								var privateMethod = function () {
									// private
								};

								var someMethod = function () {
									// public
								};

								var anotherMethod = function (text) {
									// public
									privateMethod(text);
								};

								return {
									someMethod: someMethod,
									anotherMethod: anotherMethod
								};

						})();

							// Passing date into a private method
							Module.anotherMethod('Hello!');

2. Grunt Packge

Sử dụng Grunt để build source HTML, cái này coi trong cách sử dụng Grunt/Gulp để build.

  • Sử dụng NPM
  • Sử dụng Grunt/Gulp để chạy
  • Kết hợp Live-reload (các package của các gói này kết hợp và build)
  • Có thể build SASS/LESS => CSS bình thường
  • Ngoài ra có thể install các module khác
